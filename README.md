# Appunti Uni - IBML Uniud

Questo repository contiene appunti e materiali di studio per vari corsi universitari. Gli appunti sono organizzati per materia e includono riassunti, esercizi svolti, e riferimenti utili.

## Struttura del Repository

Gli appunti sono divisi per Anno di corso, con ogni anno che contiene cartelle per ogni materia.

Tutti i file sono in formato PDF o notebook Jupyter (.ipynb) già eseguiti - sto valutando di pre esportarli come slides ma devo rivedere la struttura.

I corsi seguono un color coding specifico:

- Algoritmi e Strutture Dati — Blu (*#3498db*)
- Sistemi Operativi — Verde (*#1e7d45*)
- Statistica — Arancione (*#e67e22*)
- Fisica — Rosso (*#e74c3c*)
- TW per il cloud — Viola (*#8e44ad*)
- Machine Learning — Ciano (*#00b4d8*)
- Fondamentali di Informatica — Blu Notte (*#2c3e50*)
  
(a seguire verranno aggiunti altri corsi e colori)
